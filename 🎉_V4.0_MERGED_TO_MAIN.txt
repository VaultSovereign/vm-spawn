🚀 WHAT'S LIVE NOW

Users can now (on main branch):

1. Use MCP Server:
   ```bash
   cd ops/mcp && source .venv/bin/activate
   python remembrancer_server.py
   ```

2. Initialize Federation:
   ```bash
   ./ops/bin/remembrancer federation init
   ./ops/bin/remembrancer federation status
   ```

3. Join Federation Peers:
   ```bash
   ./ops/bin/remembrancer federation join \
     --peer http://peer.example.com:8001/mcp \
     --trust-anchor ops/certs/peer-pubkey.asc
   ```

4. Run Covenant Rituals:
   ```bash
   ./ops/bin/publish-merkle-root.sh
   ./ops/bin/attest-artifact.sh dist/artifact.tar.gz <key>
   ./ops/bin/install-hooks.sh
   ```

5. Verify All Systems:
   ```bash
   ./SMOKE_TEST.sh
   ```
